- Fetch description from individual job page if needed (optional — skip for now, use `team` + `seniority` as description).

**Site config to create in UI:**

### Shared Utilities to Build

Before Phase 2, extract these helpers into `scrapers/utils.ts`:

```ts
// Strip HTML tags from description strings
export function stripHtml(html: string): string

// Build keyword filter — returns true if job title/description matches keywords
export function matchesKeywords(text: string, keywords?: string): boolean

// Fetch with timeout + User-Agent header (already pattern in amazon.ts)
export function fetchWithTimeout(url: string, options?: RequestInit, timeoutMs = 15000): Promise<Response>
```
